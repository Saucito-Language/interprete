# Interprete Saucito

Bienvenido al repositorio oficial del lenguaje **Saucito**.

## Descripción
Saucito es un lenguaje de programación con una sintaxis inspirada en **Python**, desarrollado en **C++** y estructurado mediante el uso de tokens para el análisis léxico y sintáctico.

## Ejemplos de uso
La sintaxis de Saucito es directa y orientada a la simplicidad:

* `escribir "Hola mundo"`: Imprime texto en consola.
* `leer 2`: Captura de datos.

## Estructura del proyecto
- `Saucito v2-sin IA/`: Contiene el código fuente principal del intérprete escrito en C++.
- `x64/`: Directorio de archivos binarios generados.
